    4. 注意事项

  3. cmake 版本
    最低版本为 3.13
  4. 关闭测试
    在运行cmake时需要加上 -DENABLE_TESTING=OFF 关闭测试服务，仅构建S3时不需要运行测试，会导致后续make失败
  5. 调用动态库时报错
    在调用动态库时，报未找到CURL4之类的错误，查看curl的引用或者版本是否有误
